Script 언어란?

        기본 프로그램의 동작을 사용자의 요구에 맞게 수행되도록 해주는 용도의 언어

        별도의 소스코드 컴파일(이진수로 번역)을 하지 않고,
        Interpreter(인터프리터, 한 줄씩 코드를 읽어서 해석하는 기계, 기능)를 이용해서 코드를 바로 실행

        - 장점 : 컴파일의 소요되는 시간이 없음
        - 단점 : 프로그램 실행 시간이 상대적으로 느림

Javascript 란?

        웹 브라우저(크롬, 엣지, 웨일, 파이어폭스 등)에서
        주로 사용하는 인터프리터 방식의 객체 지향 프로그래밍 언어.

        객체 지향 프로그래밍 언어 : 객체(Object)를 만들고 조작하며, 객체끼리 
        관계를 맺음으로서 다수의 객체가 함께 명령을 수행하는 것.

Javascript 코드 작성 방법

  1. 브라우저 개발자 도구 : Console에 직접 작성(크롬 개발자 도구(F12) 켜고 콘솔 창에서 직접 작성한다
  2. HTML 내부에 script 태그를 이용해서 작성 (Internal) 작성
  3. HTML 외부에 존재하는 .js 파일을 연결해서 사용하는 방법(External 방식)
  4. 태그에 직접 JS 코드를 작성 (inline 방식, 권장 X : 코드가 지저분해짐)
